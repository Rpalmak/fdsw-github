El desarrollo web

El desarrollo web, tanto como trabajo y hobby, es apasionante y lleno de posibilidades. Conectar la creatividad con la programación y el diseño crea experiencias digitales memorables. Es un campo dinámico que permite construir aplicaciones interactivas y sitios web atractivos. El desarrollo web ofrece oportunidades emocionantes para aprender y hacer realidad ideas en el mundo virtual.

Acerca de mi

Estudios

Soy ingeniero en informática titulado en el año 2021 del INACAP. Durante mi formación académica, adquirí conocimientos sólidos en áreas como programación, bases de datos, redes y seguridad informática. Estos estudios me brindaron una base sólida para enfrentar los desafíos del campo tecnológico.

Además, tengo una certificación de inglés proficient de EFSET, lo que me permite comunicarme eficientemente en el idioma inglés. Esta certificación demuestra mi habilidad para comprender, hablar y escribir en inglés de manera fluida y precisa.

Trabajo actual

Actualmente, trabajo como desarrollador Fiori en una empresa consultora y partner de SAP. Mi principal enfoque es crear aplicaciones web y móviles utilizando las herramientas no-code proporcionadas por SAP.

Además de la creación de aplicaciones, también tengo experiencia en la implementación de RPA (Robotic Process Automation) utilizando las capacidades de automatización de SAP. Esto implica la creación de robots que realizan tareas repetitivas y tediosas, liberando así tiempo y recursos para actividades más estratégicas.

A donde voy

Sueño con ser un desarrollador web full stack. Llevo dos años inmerso en este apasionante mundo, aprendiendo constantemente a través de cursos, lecturas y un bootcamp. Mi objetivo principal es ser un desarrollador web full stack, dominando tanto el frontend como el backend.

Me esfuerzo por ser versátil en el uso de diferentes lenguajes, frameworks y herramientas para crear soluciones web innovadoras. Estoy siempre abierto a nuevas oportunidades que me permitan seguir creciendo y colaborar en proyectos web emocionantes.

Mis trabajos

Contador Virtual

El favorito de mis proyectos. Una página web para llevar la contabilidad del hogar. Tiene acceso a diferentes usuarios a través de una sesión, por lo que sirve para todos los miembros del hogar. Tiene 2 módulos activos, ingresos y egresos, donde se ingresan los valores respectivos de cada ingreso y egreso, para asi poder crear información útil de cuanto se está gastando, en que cosas se está gastando mas o menos.

Planificador de comidas

Este planificador de comidas está en proceso. Sirve para crear un menu semanal de tus 5 comidas principales según los alimentos disponibles en cada comida. Respetando además ciertas restricciones semanales, por ejemplo: no más de 3 huevos a la semana.

Web personal abogado

Página web creada para abogada, donde se muestran sus servicios, sus datos de contacto y un botón funcional para contactar a través de whatsapp.
